# media-center-on-PC-with-no-screen-with-controller-on-android-tablet
Open the setup.bat file to begin the setup process. and then follow the instructions on the bat file or in cmd currently only currently supporting windows 11 and a sourund sound setup of 5.1 on a HDMI port and you need to have vlc media player, spotify not from the microsoft store directly from [spotify.com/download](https://spotify.com/download) click Download directly from Spotify ignore the big download from the microsoft store button or download directly from [https://download.scdn.co/SpotifySetup.exe](https://download.scdn.co/SpotifySetup.exe), also need python 3.14 installed.
You will need the apk from the tablet_apk folder currently only supporting android 2.3.4 on an amazon fire kindle first generation model.
